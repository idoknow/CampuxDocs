# CampuxDocs

Campux 文档仓库。基于 [Docsify](https://docsify.js.org/) 构建。  
文档位于 docs 目录，按照 docsify 文档编写即可，写好后在 docs 目录运行 `docsify serve` 来预览。  
提交后会自动跑 GitHub Actions 构建镜像，服务器会自动拉取镜像并重启服务。  
图片放置在 `docs/assets`，引用格式请参考现有文档。
